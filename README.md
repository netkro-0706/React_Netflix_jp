## 機能  
メイン画面で人気映画リスト、評価良い映画リスト、上映予定作を見れる。  
詳細情報で映画のタイトル、評点、　ストーリー要約などを見れる。  
映画の検索が可能。  
映画の情報をSortや好きなジャンルを選択しフィルターリングが可能。
